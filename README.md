#天気と服装の提案Webアプリ#

都道府県を選択すると、Open-Meteo APIから天気情報を取得し、  
今日の天気・気温・おすすめの服装・週間予報を表示するWebアプリです。

## 制作背景
天気予報を見るだけでなく、「今日は何を着ればいいか」まで
ユーザーに分かりやすく伝えるアプリを作りたいと考え、制作しました。

## 使用技術
- HTML
- CSS
- JavaScript
- Open-Meteo API

## 主な機能
- 都道府県の選択
- 現在の天気表示
- 気温に応じた服装提案
- 週間予報の表示
- 天気や服装に合わせた画像表示

## 工夫した点
- ユーザーが直感的に使えるように、地域選択から結果表示までをシンプルな導線にしました。
- 天気情報だけでなく、服装提案まで一緒に表示することで、生活に結びつく体験を意識しました。
- 週間予報もカード形式で表示し、先の予定まで確認しやすい構成にしました。
- レスポンシブ対応を意識して、画面幅に応じて表示が崩れにくいよう調整しました。
